Perguntas frequentes

Posso trocar as imagens com facilidade?

Sim. As imagens principais são SVGs inline por desempenho. Você pode substituí-las por PNG, JPG ou SVG externos usando a tag img com loading="lazy".


Como faço teste A/B neste layout?

Há um seletor simples de variação no script. Duplique blocos de headline, CTA e herói, marque com data-variant e alterne via querystring, cookie ou flag.


Funciona sem frameworks?

Sim. É HTML, CSS e JS puros. Você pode portar para React, Next, Vue ou outro quando precisar.
